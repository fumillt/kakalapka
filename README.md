# Какалапочка: домашний рейд

Мобильная HTML5 Canvas-игра, готовая для публикации как Telegram Mini App.

## Локальный запуск

Открой `index.html` в браузере или подними любой статический сервер из корня проекта.

## Управление

- `A` / `D` или стрелки: бежать
- `W` / `Space`: прыгать. В воздухе доступен двойной прыжок.
- Возле шторы кнопка прыжка становится `Ползти`; зажми ее, чтобы карабкаться вверх.
- `E`: главное действие рядом с кошкой. Возле предмета это `Обоссать`, в пустом месте будет лужа.
- На телефоне используются большие кнопки на экране: влево, вправо, прыг/ползти, обоссать.
- В Telegram Mini App кнопки дают haptic/vibration feedback, если устройство это поддерживает.
- Шкала `Моча` тратится на предметы и лужи.
- В игре три домашних уровня без случайной генерации. На третьем уровне переноска открывается только после того, как все предметы помечены.

## Ассеты

- В меню и HUD используются локальные фотографии Какалапочки.
- Главный персонаж в уровне нарисован вручную, чтобы лучше читаться на маленьком экране.
- Предметы, для которых нет локальных фото, подгружаются как SVG из Twemoji через jsDelivr. Twemoji распространяется под CC BY 4.0.

## Публикация как Telegram Mini App

1. Выложи проект на HTTPS-хостинг: GitHub Pages, Netlify или Vercel.
2. Открой `@BotFather` в Telegram.
3. Создай бота командой `/newbot`, если его еще нет.
4. Выполни `/newapp` и выбери своего бота.
5. Укажи публичный HTTPS URL игры.
6. Добавь кнопку запуска через меню бота или команду `/setmenubutton`.

После этого игра будет открываться внутри Telegram как мобильное веб-приложение.
